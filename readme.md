## Companies table structure:

### Table structure: 

**tbl_companies**

* id
* name
* title
* description
* services
* address
* city
* province
* country
* email
* phone
* keywords

### example: 
* id: 1
* name: ABC Company
* title: Build your company app from the best in the field
* description: We are Toronto based registered company for software development
* services: web application development
* email: abc@gmail.com
* phone: 416-xxx-xxxx
* address: 333 Yonge street
* city: Toronto
* Province: Ontario
* Country: Canada
* Keywords: web, applications
